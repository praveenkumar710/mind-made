## Quick Start

### 1. Clone and Install

\`\`\`bash
git clone <your-repo-url>
cd mindmate
npm install
\`\`\`

### 2. Environment Setup

\`\`\`bash
npm run setup
\`\`\`

This creates a `.env.local` file with secure defaults. Update it with your API keys:

\`\`\`env
# Required
MONGODB_URI=mongodb://localhost:27017/mindmate
JWT_SECRET=<auto-generated>

# Optional (for full functionality)
OPENAI_API_KEY=sk-...
XAI_API_KEY=xai-...
TWILIO_ACCOUNT_SID=AC...
TWILIO_AUTH_TOKEN=...
TWILIO_PHONE_NUMBER=+1...
\`\`\`

### 3. Database Setup

**Option A: Local MongoDB**
\`\`\`bash
# Install MongoDB locally
brew install mongodb/brew/mongodb-community  # macOS
# or follow MongoDB installation guide for your OS

# Start MongoDB
brew services start mongodb/brew/mongodb-community
\`\`\`

**Option B: MongoDB Atlas (Recommended)**
1. Create account at [MongoDB Atlas](https://cloud.mongodb.com/)
2. Create a free cluster
3. Get connection string and update `MONGODB_URI` in `.env.local`

### 4. Run Development Server

\`\`\`bash
npm run dev
\`\`\`

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Configuration

### Required Environment Variables

- `MONGODB_URI`: MongoDB connection string
- `JWT_SECRET`: Secret for JWT token signing (auto-generated)

### Optional Environment Variables

- `OPENAI_API_KEY`: For AI chat functionality
- `XAI_API_KEY`: For Grok AI integration
- `TWILIO_*`: For SMS OTP authentication
- `NEXTAUTH_*`: For enhanced authentication features

### API Keys Setup

**OpenAI API Key:**
1. Visit [OpenAI Platform](https://platform.openai.com/)
2. Create API key
3. Add to `.env.local`

**Grok/XAI API Key:**
1. Visit [XAI Console](https://console.x.ai/)
2. Create API key
3. Add to `.env.local`

**Twilio (for SMS):**
1. Create [Twilio account](https://www.twilio.com/)
2. Get Account SID, Auth Token, and Phone Number
3. Add to `.env.local`
